Liftsuggest is a product recommendation system based on your Google analytics data. It works with WP - Ecommerce plugin.

== Description ==

Liftsuggest is a product recommendation tool that helps generate recommended products.Based on the products users are viewing or adding to the shopping cart, Liftsuggest dynamically generates top upsell/cross sell recommendations.

This service use Google Analytics data to generates recommendations based on it.

You can modify various module-specific attributes such as user token, user ID, domain name etc. from the admin panel for modules. 

Please feel free to reach out to us at info@liftsuggest.com for any queries related to LiftSuggest Recommendations.


Please Note:
- You must need installed "WP Shopping Cart" Plugin.(Tested up to: Version 3.7.5.3 of WP Shopping Cart)
- You'll need to register with www.liftsuggest.com in order to use this plugin. 
- To Know More Visit: www.liftsuggest.com


== Installation ==

1. Upload the 'liftsuggest' folder to the '/wp-content/plugins/' directory
2. Activate the Plugin through the 'Plugins' menu in Wordpress Admin
3. You must set module-specific attributes such as user token, user ID, domain name in admin panel to Enable module.
4. If you are using  traditional google analytics tracking  please include the following code before pageTracker._trackPageview() call and after var pageTracker = _gat._getTracker("UA-xxxxxx-x") in your template (ideally it is YOURSTOREPATH\wordpress\wp-content\themes\twentyeleven\footer.php)

== Frequently Asked Questions ==

= Why should I use Liftsuggest"? =

If you run e-commerce store and considering how to increase your revenue, Liftsuggest is your answer !! Liftsuggest can generate the product recommendations on your site by just using your Google Analytics data. Simple as it sounds ! You just need to sign up with Liftsuggest.Its free!

= From where I can get "user token, user ID, domain name"? =

Once you register with http://www.LiftSuggest.com , you will receive email with authentication token & other information for you to start using the plugin. 

= Why do I need to register with LiftSuggest? =

Liftsuggest Algorithm needs your Google analytics e-commerce data to generate product recommendations.

= How my data is secured with LiftSuggest? =

LiftSuggest "doesnt" ask for your Google Analytics Password. During registration you will be using highly secure OAuth to allow Liftsuggest to extract google analytics data of your e-commerce store/webshop.
